# Product Selection Criteria

## Hard Constraints (must have)

1. **Buildable by AI agent** — продукт можно создать силами одного AI-агента (код, контент, стратегия)
2. **Low/zero initial capital** — не требует начального капитала, серверов, лицензий
3. **Digital-first** — полностью цифровой продукт, без физических компонентов
4. **Solo-friendly** — не требует команды на старте

## Scoring Criteria (0-10)

| Criterion | Weight | Description |
|-----------|--------|-------------|
| Market Size | 3x | TAM > $10B для потенциала $1B |
| Timing | 3x | Рынок созрел, но не перенасыщен |
| AI Advantage | 2x | AI-агент может делать это лучше/быстрее людей |
| Viral/Network Effects | 2x | Органический рост без бюджета на маркетинг |
| Revenue Path | 2x | Понятный путь к монетизации |
| Moat | 1x | Защита от копирования (data, network, brand) |
| Incrementality | 1x | Можно запустить MVP за 1-2 недели сессий |

## Anti-patterns (avoid)

- "Yet another chatbot" — перенасыщенный рынок
- B2C consumer apps — дорогое привлечение
- Продукты требующие GPU/инфраструктуру на старте
- Marketplace/platform без ответа на chicken-and-egg
