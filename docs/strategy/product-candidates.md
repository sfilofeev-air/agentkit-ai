# Product Candidates

## Candidate 1: AI Agent Framework (Open Source)
**Idea:** Open-source фреймворк для создания AI-агентов с монетизацией через cloud/enterprise.
- Market: $50B+ (AI infrastructure)
- Precedent: LangChain ($10B+), CrewAI
- Pros: viral growth через developers, moat через community
- Cons: очень конкурентный, нужно быть технически лучше
- AI Advantage: AI-агент строит лучший фреймворк для AI-агентов (мета-преимущество)
- Score: pending

## Candidate 2: Vertical AI SaaS (Legal/Compliance)
**Idea:** AI-инструмент для автоматизации compliance/legal для SMB.
- Market: $30B+ (legal tech + compliance)
- Precedent: Harvey AI ($1.5B), Ironclad
- Pros: высокая willingness to pay, sticky product
- Cons: нужна доменная экспертиза, sales cycle длинный
- AI Advantage: AI может генерировать шаблоны, анализировать документы
- Score: pending

## Candidate 3: AI Content Engine / Media Company
**Idea:** AI-driven медиа-компания: автоматическая генерация контента, newsletters, social media presence. Монетизация: реклама, подписки, данные.
- Market: $100B+ (digital media + advertising)
- Precedent: Morning Brew ($75M acq), The Hustle
- Pros: AI может генерить контент 24/7, viral potential
- Cons: долго строить аудиторию, сложно дифференцироваться
- AI Advantage: бесконечное производство контента без людей
- Score: pending

## Candidate 4: AI-Powered Dev Tools Platform
**Idea:** Платформа, которая автоматизирует рутинные dev tasks: code review, documentation, testing, migrations.
- Market: $45B+ (developer tools)
- Precedent: GitHub Copilot, Cursor, Devin
- Pros: developers платят, product-led growth
- Cons: конкуренция с Big Tech (Microsoft, Google)
- AI Advantage: AI-агент лучше всех знает как AI-агенту удобно работать
- Score: pending

## Candidate 5: AI Workflow Automation (Zapier killer)
**Idea:** No-code AI автоматизация бизнес-процессов. AI сам понимает что нужно автоматизировать и делает это.
- Market: $20B+ (workflow automation)
- Precedent: Zapier ($5B), Make, n8n
- Pros: понятный revenue model, sticky, network effects через templates
- Cons: интеграции — основная работа, нужны API ключи
- AI Advantage: AI может сам создавать workflows по описанию
- Score: pending

## Candidate 6: AI Data Analytics Platform
**Idea:** "Спроси свои данные на естественном языке" — подключаешь БД/CSV/API и получаешь insights.
- Market: $30B+ (BI & analytics)
- Precedent: ThoughtSpot, Mode
- Pros: clear value prop, every company needs this
- Cons: data security concerns, нужны интеграции
- AI Advantage: AI отлично генерит SQL и визуализации
- Score: pending
